# Morse-Translator
A simple Morse Code to Text / Text to Morse Code Translator

## Features

- Convert plain text to Morse code.
- Convert Morse code to plain text.
- Supports letters, numbers, and basic punctuation.

## Usage

1. **Clone the repository:**
   ```
   git clone https://github.com/yourusername/Morse-Translator.git
   cd Morse-Translator
   ```

2. **Run the translator:**
     - Linux/Mac/Termux:
        ```
        python3 main.py
        ```
     - Windows:
        ```
        python main.py
        ```

3. **Follow the prompts to enter text or Morse code.**

## Example

- **Text to Morse:**
  ```
  Input: Hello World
  Output: .... . .-.. .-.. --- / .-- --- .-. .-.. -..
  ```

- **Morse to Text:**
  ```
  Input: .... . .-.. .-.. --- / .-- --- .-. .-.. -..
  Output: Hello World
  ```

## Requirements

- Python 3.x
